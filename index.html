Welcome to OpenSupports.

OpenSupports is an open source support ticket system written in PHP. It's complety free and it is under MIT License.

If you have a website and you wish to bring help to your customers. You may think of making a contact email of a contact form. But the problem is when you need a better administration of users' consults. You can not have an unique contact email to reply everything.

The solution is OpenSupports: OpenSupports is a support system you can easily implement into your website.

How it works

OpenSupports works in a very simple way: 1. You install OpenSupports into your website 2. Users have questions, let's call them "ticekts". 3. You or your staff will answer users' ticekts. 4. The users send their ticekts through OpenSupports 5. OpenSupports decides what path should the ticket take and then send it to the staff.

For example, you can have many categories or departments to determinate the path of the ticket.

As an example, if an user has a suggestion. OpenSupport will send that ticket to suggentions' department. If the user have an issue with your product, the ticket will be sent to issues' department. If you bring multilanguage support, you can have several languages to make a better path. You can also add more filters to tickets and users, like Country or Priority.

Why OpenSupports?

OpenSupports has been written to be flexible, this means you can use an users system (where every user has a profile) or don't use it and use a form to send ticket without users registration. You can also disable registration but still using a users system. This allows you to put registration in somewhere else or use it for exclusive costumers.

One of the best features OpenSupports 3 is that it is an open source script. This means you can modify it to make it fit into your proyect. Even if you want to use for commercial applications, you can do it because it's under MIT License.

Plugins API

OpenSupports 3 has a totally new plugins' API, you can make lots of modifications and plugins. You can also create new styles based in CSS.

Even if you are not a developer, you can easily install plugins and styles. There is a huge database of plugins and styles that are enable for download.

At the OpenSupports' website you can find an amount of tutorials and video tutorials to get the best result. There's also a wiki which includes development documentation and a forum for the community.
